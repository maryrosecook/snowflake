Snowflake.js

 by mary rose cook
Run code in beautiful and unique situations

What is this library?

Snowflake lets you run a piece of code once, or every two seconds, or when a value changes, or when it stops changing. It is useful for debugging a function that is run frequently.

The basic idea

There are four functions in Snowflake: once(), every(), stuck() and change(). They are all used in the same basic way: First, you make an instance of Snowflake. Second, you call a snowflake function on the instance. Third, in some cases, you pass arguments that the snowflake function needs (like an interval for the every() function). Fourth, you pass the action function to run. Fifth, and optionally, you pass an argument that will get passed into the action function.

once()

optionalArgument gets passed into the action function.

every()

interval is an integer and is in seconds.
optionalArgument gets passed into the action function.

stuck()

The first argument is an array of variables to watch.
If watchVar1 or watchVar2 have the same value as the previous go round, the action function is run.
optionalArgument gets passed into the action function.

change()

The first argument is an array of variables to watch.
If watchVar1 or watchVar2 have a different value to the previous go round, the action function is run.
optionalArgument gets passed into the action function.
